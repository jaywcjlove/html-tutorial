HTML dir 属性
===

## 定义和用法

`dir` 属性指定元素内容的文本方向。

## 适用于

`dir` 属性是一个 [全局属性](../reference/standardattributes.md)，可用于任何 HTML 元素。

| 元素 Element | 属性 Attribute |
| ----- | ----- |
| 全部 [HTML](../tags/README.md) 元素 | [dir](./global/dir.md) |
<!--rehype:style=width: 100%; display: inline-table;-->

## 示例

具有从右到左方向的段落：

```html idoc:preview:iframe
<p dir="rtl">
  从右到左写这个文本！
</p>
```
