| [\<title>](../tags/title.md) | 定义文档的标题 |
| [\<body>](../tags/body.md) | 定义文档的主体。 |
| [\<h1> to &lt;h6>](../tags/hn.md) | 定义 HTML 标题 |
| [\<p>](../tags/p.md) | 定义段落。 |
| [\<br>](../tags/br.md) | 定义单个换行符 |
| [\<hr>](../tags/hr.md) | 定义水平线。 |
| [\<!--...-->](../tags/comment.md) | 定义注释 |
