| [&lt;title&gt;](../tags/title.md) | 定义文档的标题 |
| [&lt;body&gt;](../tags/body.md) | 定义文档的主体。 |
| [&lt;h1&gt; to &lt;h6&gt;](../tags/hn.md) | 定义 HTML 标题 |
| [&lt;p&gt;](../tags/p.md) | 定义段落。 |
| [&lt;br&gt;](../tags/br.md) | 定义单个换行符 |
| [&lt;hr&gt;](../tags/hr.md) | 定义水平线。 |
