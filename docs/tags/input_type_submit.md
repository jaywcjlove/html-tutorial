HTML \<input type="submit">
===

## 示例

定义提交按钮：

```html idoc:preview:iframe
<input type="submit">
```

## 定义和用法

`<input type="submit">` 定义了一个提交按钮，它将所有表单值提交给表单处理程序。

表单处理程序通常是带有用于处理输入数据的脚本的服务器页面。

表单处理程序在表单的 `action` 属性中指定。
