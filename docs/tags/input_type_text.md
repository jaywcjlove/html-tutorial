HTML \<input type="text">
===

## 示例

定义一个用户可以输入文本的单行文本字段：

```html idoc:preview:iframe
<label for="fname">First name:</label>
<input type="text" id="fname" name="fname">
```

## 定义和用法

`<input type="text">` 定义了一个单行文本字段。

文本字段的默认宽度为 20 个字符。

**提示：** 始终添加 `<label>` 标签以获得最佳可访问性实践！
