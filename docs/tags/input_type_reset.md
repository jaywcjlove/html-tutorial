HTML \<input type="reset">
===

## 示例

定义一个重置按钮：

```html idoc:preview:iframe
<input type="reset">
```

## 定义和用法

`<input type="reset">` 定义了一个重置按钮，它将所有表单值重置为其初始值。

**提示：** 避免表单中的重置按钮！ 如果用户错误地点击它们，这会让用户感到沮丧。

| Attribute | ![chrome][1] | ![edge][2] | ![firefox][3] | ![safari][4] | ![opera][5] |

## 语法

