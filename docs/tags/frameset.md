HTML \<frameset> 标签
===

[![](https://shields.io/badge/HTML5-已废弃-yellow?logo=HTML5)](https://caniuse.com/?search=<frame>)

## HTML5 不支持

`<frameset>` 标签在 HTML 4 中用于定义框架集。

## 改用什么？

使用 [\<iframe>](./iframe.md) 标签在当前 HTML 文档中嵌入另一个文档：

```html idoc:preview
<iframe src="./iframe.html" width="100%" height="90%"></iframe>
```
