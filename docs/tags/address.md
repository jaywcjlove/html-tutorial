HTML `<address>` 标签
===

## 例子

Example.com 的联系信息：

```html idoc:preview
<address>
Written by <a href="mailto:webmaster@example.com">Jon Doe</a>.<br>
Visit us at:<br>
Example.com<br>
Box 564, Disneyland<br>
USA
</address>
```

## 定义和使用

`<address>` 标记定义文档或文章的作者/所有者的联系信息。

联系信息可以是电子邮件地址、URL、物理地址、电话号码、社交媒体句柄等。

`<address>` 元素中的文本通常以斜体呈现，浏览器总是会在 `<address>` 元素之前和之后添加换行符。
