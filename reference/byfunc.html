HTML 标签参考 (分类)

按功能类别排列 HTML 标签

基本 HTML

标签 Tag描述 Description
<!DOCTYPE>定义文档类型
<html>定义 HTML 文档的根
<head>包含文档的元数据/信息
<title>定义文档的标题
<body>定义文档的主体。
<h1> to <h6>定义 HTML 标题
<p>定义段落。
<br>定义单个换行符
<hr>定义水平线。
<!--...-->定义注释

格式化

标签 Tag描述 Description
<abbr>定义缩写或首字母缩略词
<acronym> ⚠️ HTML 5 不支持。请改用 <abbr>。定义首字母缩写词
<address>定义文档作者/所有者的联系信息
<b>定义文本粗体字。
<bdo>定义文字方向。
<big> ⚠️ HTML 5 不支持。 定义大号文本。请改用 CSS 代替。
<blockquote>定义长的段落引用。
<center> ⚠️ HTML 5 不支持。改用 CSS。 定义居中文本
<cite>定义作品的标题
<code>定义一段计算机代码
<del>定义已从文档中删除的文本
<dfn>指定将在内容中定义的术语
<em>定义强调的文本
<font> ⚠️ HTML 5 不支持。改用 CSS。 定义文本的字体、颜色和大小
<i>定义斜体字。
<ins>定义被插入文本。
<kbd>定义键盘文本。
<mark>定义标记/突出显示的文本
<meter>定义已知范围内的标量测量（仪表）
<pre>定义预格式化文本
<progress>代表一个任务的进度
<q>定义一个短引号
<rp>定义在不支持 ruby 注释的浏览器中显示的内容
<rt>定义字符的解释/发音（用于东亚排版）
<ruby>定义一个 ruby 注释（用于东亚字体）
<s>定义不再正确的文本
<samp>定义计算机代码样本
<small>定义较小的文本
<strike> ⚠️ HTML 5 不支持。请改用 <del> 或 <s>。 定义删除线文本
<strong>定义重要文本
<sub>定义下标文本
<sup>定义上标文本
<template>为页面加载时应该隐藏的内容定义一个容器
<time>定义特定时间（或日期时间）
<tt> ⚠️ HTML 5 不支持。改用 CSS。定义打字机文本
<u>定义下划线文本
<var>定义一个变量
<wbr>定义可能的换行符

表单和输入

标签 Tag描述 Description
<form>定义用户输入的 HTML 表单
<input>定义输入控件。
<textarea>定义一个多行输入控件（文本区域）
<button>定义一个可点击的按钮
<select>定义一个下拉列表
<optgroup>在下拉列表中定义一组相关选项
<option>在下拉列表中定义一个选项
<label>为 <input> 元素定义标签
<fieldset>对表单中的相关元素进行分组
<legend>定义 <fieldset> 元素的标题
<datalist>指定输入控件的预定义选项列表
<output>定义计算结果

框架

标签 Tag描述 Description
<frame> ⚠️ HTML 5 不支持。在框架集中定义一个窗口（一个框架）
<frameset> ⚠️ HTML 5 不支持。 定义一组框架
<noframes> ⚠️ HTML 5 不支持。为不支持框架的用户定义替代内容
<iframe>定义一个内联框架

图像

标签 Tag描述 Description
<img>定义图像。
<map>定义图像映射
<area>定义图像映射内的区域。
<canvas>用于通过脚本（通常是 JavaScript）即时绘制图形
<figcaption>定义 <figure> 元素的标题
<figure>指定自包含内容
<picture>定义多个图片资源的容器
<svg>定义 SVG 图形的容器

音频/视频

标签 Tag描述 Description
<audio>定义声音内容。
<source>为媒体元素（<video> 和 <audio>）定义多种媒体资源
<track>为媒体元素（<video> 和 <audio>）定义文本轨道
<video>定义嵌入的视频内容

链接

标签 Tag描述 Description
<a>定义超链接
<link>定义文档和外部资源之间的关系（最常用于链接到样式文件）
<nav>定义导航链接

列表

标签 Tag描述 Description
<ul>定义一个无序列表
<ol>定义有序列表
<li>定义一个列表项
<dir> ⚠️ HTML 5 不支持。 请改用 <ul>。定义目录列表
<dl>定义描述列表
<dt>在描述列表中定义一个术语/名称
<dd>定义描述列表中术语的描述/值

表格

标签 Tag描述 Description
<table>定义一个表
<caption>定义表格标题
<tr>定义表中的一行
<th>定义表格中的标题单元格
<td>定义表格中的单元格
<thead>对表格中的标题内容进行分组
<tbody>将正文内容分组到表格中
<tfoot>对表格中的页脚内容进行分组
<col>为 <colgroup> 元素中的每一列指定列属性
<colgroup>指定表格中的一组一个或多个列进行格式化

样式和语义

标签 Tag描述 Description
<style>定义文档的样式信息
<div>定义文档中的一个部分
<span>定义文档中的一个部分
<header>定义文档或部分的标题
<footer>定义文档或部分的页脚
<main>指定文档的主要内容
<section>定义文档中的一个部分
<article>定义文章
<aside>定义页面内容之外的内容。
<details>定义用户可以查看或隐藏的其他详细信息
<dialog>定义一个对话框或窗口
<summary>定义 <details> 元素的可见标题
<data>添加给定内容的机器可读翻译

元信息

标签 Tag描述 Description
<head>包含文档的元数据/信息
<meta>定义有关 HTML 文档的元数据
<base>指定文档中所有相对 URL 的基本 URL/目标
<basefont> ⚠️ HTML 5 不支持。定义页面中文本的默认字体、颜色或尺寸。

编程

标签 Tag描述 Description
<applet> ⚠️ HTML 5 不支持。请改用 <embed> 或者 <object>。定义一个嵌入式小程序。
<bdi>定义文本的文本方向，使其脱离其周围文本的方向设置。
<embed>为外部应用程序定义一个容器
<noscript>为不支持客户端脚本的用户定义替代内容
<object>为外部应用程序定义一个容器
<param>定义对象的参数
<script>定义客户端脚本
