HTML 字符集实体

HTML5 规范鼓励 Web 开发人员使用 UTF-8 字符集！情况并非总是如此。 早期网络的字符编码是 ASCII。

后来，从 HTML 2.0 到 HTML 4.01，ISO-8859-1 被认为是标准字符集。

随着 XML 和 HTML5，UTF-8 终于到来，解决了很多字符编码问题。
